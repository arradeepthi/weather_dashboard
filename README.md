## Weather Dashboard
Weather dashboard is a website that provides real-time weather data and a 5-day forecast for any city.

**[Live Demo](https://arradeepthi.github.io/weather_dashboard/)**

## Tech Stack:
1. HTML<br>
2. CSS<br>
3. JavaScript<br>
4. Bootstrap<br>
5. OpenWeatherMap API<br>
